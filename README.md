# 搜索结果始终在新标签页打开

一个 Tampermonkey 用户脚本。使用鼠标左键点击搜索结果时，自动在新标签页中打开链接，无需中键或 `Ctrl + 点击`。

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 或其他兼容的用户脚本管理器。
2. 点击下面的链接，并在用户脚本管理器中确认安装：

### [点击安装脚本](https://raw.githubusercontent.com/PaulHerrero/search-results-new-tab/main/search-results-new-tab.user.js)

## 支持的搜索引擎

Google、Bing、百度、DuckDuckGo、Yahoo、搜狗、Yandex、Ecosia、Startpage、Brave Search。

## 功能

- 左键点击搜索结果时在新标签页打开
- 保留修饰键的原有行为
- 不影响搜索结果分页和导航链接
- 可切换后台打开和紧邻当前标签页插入
